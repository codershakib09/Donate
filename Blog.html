Important FAQ

What is the Document Object Model(DOM)?

The DOM (Document Object Model) is a programming interface for web documents. It represents the structure of a document as a tree of objects.This tree-like structure allows each part of the document to be accessed and manipulated programmatically.

How do you select an element from the DOM?

I have many ways to select an element from the DOM: document.getElementsByTagName() is used to select elements by their tag name. document.getElementById() is used to select an element by its unique id attribute. document.querySelector() allows to select the first element that matches a CSS selector. document.querySelectorAll() selects all elements that match a CSS selector and returns a NodeList. There are more ways to select elements in the DOM.

What is event delegation in the context og the DOM, and whuy it is useful?

Event delegation is a technique where a single event listener is added to a parent element instead of individual child elements. The event then "bubbles up" to the parent, leveraging DOM event propagation for efficient handling.

How do you manipulatale an elements attributes and styles using the DOM?

i can manipulate an element's attributes using getAttribute(), setAttribute(), and removeAttribute(). For styles, i can change CSS directly with element.style. such as element.style.backgroundColor = 'red' .And also i can use classList.add or remove.
